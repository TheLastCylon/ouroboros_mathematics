# Ouroboros Mathematics
### A Consistent Framework for Division by Zero
*Dirk J. Botha, April 2026*

Standard arithmetic declares `0 ÷ 0` undefined and moves on.
This framework does not move on.

## What this is

A formal mathematical framework built on a single distinction that
standard notation ignores: division and ratio are not the same operator.
Division is operational. Ratio is relational. They can agree — but they
are not equivalent, and treating them as equivalent fails catastrophically
at the boundary.

From three axioms and one named object (`⟲ = 0 ÷ 0`), the framework
derives a system closed under all division by zero, a proof that
`0 : 0 = 1`, and a singularity formula showing that inside `⟲`, no
operation distinguishes any value from any other.

The entire thing follows from refusing to discard the nine in `0 : 9`.
